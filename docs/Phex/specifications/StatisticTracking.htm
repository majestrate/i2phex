The Phex Global Statistic Tracking

for Phex 0.X

Requirements

The current Phex statistic tracking is very week therefor there will be a completly new statstic tracking concept implemented. The new concept should address the following issues:
  • Offer a easyly extendable way to track any statistic that might come up in the future.
  • Statistics are displayed to the user in a table.
  • Allows the user to select the statistics it likes to track/display from a list. Some statistics that are not displayed might not get tracked and others that are not displayed are still tracked. This depends on the type of statistic. And how expensive the tracking is.
  • For more performant user interface updates statistics are pooled from the underlying data storage in a defined cycle of 1 to 60 seconds.
