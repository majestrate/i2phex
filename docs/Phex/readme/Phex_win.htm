Introduction

Phex is a free, open source, multi-platform Gnutella client with no spy- or adware. Phex supports the latest Gnutella protocol and proposals like multi-source downloads (swarming), automated searches, browse host, GWebCache and many more.

Latest Release

You can download the latest release here:
http://phex.org/

System Requirements

To use Phex you need a JavaTM 2 Virtual Machine 1.4 or higher installed.

Installing Phex

To install Phex on Windows just start the Phex self-installer and follow the instruction of the installation dialog.

Starting Phex

During installation Phex creates a program group in your Windows start menu. Start Phex by clicking on the program icon in the Phex program group.
